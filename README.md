# Portfolio2
Web project UTS untuk matakuliah Pemrograman Berbasis Platform menggunakan HTML CSS dan JS dengan bantuan Framework TailwindCSS v4.x
Web ini dibuat dengan 3 file .html yang berbeda yang dimana setiap file memiliki fungsinya masing-masing
Web ini juga memiliki tampilan dark mode dan juga light mode
